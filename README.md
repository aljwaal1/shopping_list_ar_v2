# قائمة المشتريات

تطبيق Flutter Android خفيف ضمن قالب APK المباشر.

## المزايا

- واجهة عربية RTL.
- تصميم V2 محسّن وهوية مختلفة.
- حفظ محلي بسيط باستخدام shared_preferences.
- أصوات خفيفة من النظام بدون مكتبات صوت إضافية.
- صفحة مراسلة المطور بزر نسخ الرسالة فقط.
- بناء APK مباشر عبر GitHub Actions.

## APK

سيظهر الملف النهائي بعد نجاح البناء داخل:

```text
apk/shopping-list.apk
```
