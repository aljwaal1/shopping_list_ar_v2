# Google Play Store Listing Draft

## Arabic

### App name
قائمة المشتريات

### Short description
قائمة تسوق عربية بسيطة وسريعة تعمل بدون إنترنت.

### Full description
قائمة المشتريات تطبيق عربي بسيط يساعدك على تنظيم مشترياتك اليومية بسرعة ووضوح.

يمكنك إضافة العناصر، تحديد الكمية، اختيار التصنيف، تعليم العناصر كمكتملة، البحث داخل القائمة، ونسخ القائمة لمشاركتها أو حفظها.

المميزات:
- واجهة عربية RTL سهلة وواضحة.
- إضافة سريعة لعناصر التسوق.
- تصنيفات مثل بقالة، ألبان، خضار، منزل، وأخرى.
- متابعة العناصر المتبقية والمكتملة.
- بحث داخل القائمة.
- نسخ القائمة كنص.
- يعمل بدون إنترنت.
- لا يحتاج تسجيل حساب.
- لا يطلب صلاحيات حساسة.

التطبيق مناسب للاستخدام اليومي في البيت، السوق، أو أثناء التخطيط للمشتريات.

## English

### App name
Arabic Shopping List

### Short description
A simple Arabic shopping list app that works offline.

### Full description
Arabic Shopping List is a simple and clean app for organizing daily shopping needs.

You can add items, set quantities, choose categories, mark items as completed, search your list, and copy the list as text.

Features:
- Arabic RTL interface.
- Quick item entry.
- Shopping categories.
- Pending and completed item tracking.
- Search inside the list.
- Copy list as text.
- Works offline.
- No account required.
- No sensitive permissions required.

The app is designed for simple daily use at home, in the market, or while planning purchases.
